App Overview

Section 02

Installing the App

DebateMind is a Progressive Web App — it can be installed directly from the browser without visiting any app store.

Android (Chrome)
📱 Install on Android Phone
  1. Open the app in Google Chrome on your Android phone.
  2. Wait a few seconds — an "Install DebateMind" banner will appear at the bottom.
  3. Tap "Install".
  4. The app will appear on your Home Screen like a native app.
  5. Alternatively: tap the three-dot menu (⋮) → "Add to Home Screen".
Desktop (Windows / Mac / Linux)
🖥 Install on Desktop
  1. Open the app in Google Chrome or Microsoft Edge.
  2. Look for the install icon (⊕) in the address bar on the right side.
  3. Click it and select "Install".
  4. The app will open in its own window, separate from the browser.
💡

Offline Use: After the first visit, DebateMind works completely offline. All your topics, notes, and progress are stored on your device and never sent to any server.

Running Locally (For Teachers / Developers)

  1. Place all files (index.html, manifest.json, service-worker.js, author.jpg) in one folder.
  2. Open VS Code → install Live Server extension → right-click index.html → "Open with Live Server".
  3. Or use Python: python -m http.server 8000 then visit localhost:8000.
  4. Service worker (offline) only works on localhost or HTTPS — not file:// protocol.

Topic Generator Tab 🎯

This is the core module. It generates debate topics from a curated database of 100+ topics aligned with NCF 2023 thematic learning areas.

Category Filter

Select the subject area for your topic. Options: All Topics, Technology, Education, Environment, Society, Ethics. Each category maps to real-world issues relevant to students.

Difficulty Selector

LevelSuitable ForExample
🟢 BeginnerClasses 6–8, first-time debaters"Mobile phones should be allowed in classrooms."
🟡 IntermediateClasses 9–10, some debate experience"Standardised examinations fail to measure true student potential."
🔴 AdvancedClasses 11–12, competitive debate"NCF 2023's holistic approach is more effective than subject-based instruction."

Generating a Topic

  • 1
    Select Category & Difficulty
    Choose your desired category (or All) and difficulty level before generating.
  • 2
    Press "Generate New Topic"
    A random topic from the selected parameters appears. The topic card shows category, topic text, and difficulty.
  • 3
    Action Buttons
    Save Topic → stores it in your saved list. Copy Topic → copies to clipboard. Use in Arguments → sends it to the Argument Builder.
  • 4
    Saved Topics List
    Appears below the generator. Saved topics persist across sessions (stored in device memory). Delete any saved topic with the ✕ button.
🎓

Classroom Use: Teachers can project the app on a screen, generate a topic, and use it as the starting point for a class debate. Use the "Use in Arguments" button to immediately move to structured argument building.

Section 05

Argument Builder Tab 💡

This module helps students structure their debate arguments using guided prompts. It supports the NCF 2023 emphasis on reflective thinking and analytical reasoning.

Topic Input

Type your debate topic in the text field, or use "Use in Arguments" from the Topic Generator to pre-fill it automatically.

Side Selection

Click PRO / FOR (supporting side, teal highlight) or CON / AGAINST (opposing side, red highlight). The argument prompts change based on your selection.

Argument Prompts

Six guided prompts appear in a grid. Each prompt is a question that scaffolds a different aspect of debate preparation:

PromptPurpose (PRO side)Purpose (CON side)
01List 3 strongest arguments FORList 3 strongest arguments AGAINST
02Evidence & real-world examplesContradicting evidence
03Benefits to societyRisks and dangers
04What would change for the betterWho is harmed
05Responding to the main objectionAlternatives to the opposing view
06Supporting values (freedom, progress…)Opposing values (caution, equity…)

Click any prompt card to add it as a heading in your Notes area. Then write your argument beneath it.

Notes Area

A large text area for writing out your complete argument. Use the prompts as scaffolding. The Save Notes button stores your notes (up to 20 entries) on your device. Copy copies to clipboard. Clear empties the area.

⚠️

Important: Notes are saved per session. Clearing your browser data or using a different device will not show previous notes. Advise students to copy important notes to a text file or notebook.

Section 06

Debate Timer Tab ⏱

The Timer module simulates real debate conditions with structured, multi-phase sessions. Practicing under time constraints is essential to developing disciplined speaking skills.

Timer Modes

ModePreparationSpeechRebuttalBest For
⚡ Quick30 sec1 min30 secWarm-up, beginners
📢 Standard1 min2 min1 minRegular classroom use
🎓 Extended2 min3 min2 minAdvanced, competitive practice
⚙ CustomUser-definedUser-definedUser-definedTeacher-specific needs

Using the Timer

  • 1
    Select Timer Mode
    Click the mode button matching your session type. For Custom, enter times in seconds in the fields that appear.
  • 2
    Press ▶ Start
    The circular progress ring begins counting down. The phase name (PREPARATION / SPEECH / REBUTTAL) shows in the centre.
  • 3
    Phase Transitions
    When one phase ends, a toast notification announces the next phase. The ring colour changes per phase (blue → gold → teal).
  • 4
    Pause / Reset
    ⏸ Pause freezes the timer. ↺ Reset returns to the first phase. Useful between speakers in a class debate.

The Three Phases Explained

Preparation Phase: Student silently organises their thoughts and notes. No speaking. Use the Argument Builder before this phase.

Speech Phase: Student delivers their structured argument. Encourage them to state their position, give evidence, and conclude.

Rebuttal Phase: Student responds to the opposing argument. Focus on addressing specific points raised, not repeating the speech.

🎓

Classroom Use: Project the timer on the board. Each student runs through all three phases. During preparation phase, others can also note down counterarguments using the Argument Builder.

Section 07

Pro vs Con Tab ⚖️

This module randomly assigns students both a debate topic AND a side (Pro or Con). Defending a randomly assigned position builds perspective-taking ability — a core competency in both NCF 2023 and PARAKH assessments.

How to Use

  • 1
    Press "Assign Me a Topic & Side"
    The app randomly selects a topic from any category and difficulty, then randomly assigns PRO or CON.
  • 2
    Read Your Assignment
    The topic appears in large text. A coloured badge shows your side: teal = PRO, red = CON. Instructions explain your goal.
  • 3
    Use Brainstorm Prompts
    Four prompts suited to your assigned side appear below. Click any prompt to add it to your notes area.
  • 4
    Write Your Response
    Use the notes area to draft your argument. You have full freedom — use the timer for time pressure.
  • 5
    Save or Get New Challenge
    "Save" stores your notes. "New Challenge" generates a completely fresh assignment.
💡

Advanced Use: Have two students press "Assign Me a Topic" simultaneously. If they get the same topic but opposite sides, they are ready for a live two-person debate! Combine with the Debate Timer for a complete structured session.

Why Random Side Assignment Matters

Research consistently shows that students who practice defending positions they personally disagree with develop stronger critical thinking and more nuanced understanding of complex issues. This is directly aligned with NCF 2023's emphasis on reflective and perspective-aware learning and NEP 2020's call for open-minded inquiry.

Section 09

Using DebateMind in the Classroom

DebateMind is designed to support both individual student practice and structured classroom debate activities.

Suggested Classroom Session Plan (45 minutes)

  • 1
    Warm-Up: Topic Discovery (5 min)
    Teacher projects app on board. Generate 3–4 topics using the Topic Generator. Students vote on which to debate.
  • 2
    Research & Prep: Argument Builder (10 min)
    Divide class into Pro and Con groups. Each student opens Argument Builder on their device, enters the topic, selects their side, and uses prompts to build notes.
  • 3
    Individual Debate Practice: Timer (15 min)
    Each group sends 2–3 representatives. Run Standard Timer (1+2+1 min) for each speaker. Remaining students note counterarguments.
  • 4
    Pro vs Con Challenge (10 min)
    Select 4 students. Use Pro vs Con mode to assign random topics and sides. Give them 2 minutes to prepare using brainstorm prompts, then 2 minutes to present.
  • 5
    Reflection (5 min)
    Discuss: What was the strongest argument heard? What evidence was most convincing? How does this connect to NEP 2020's critical thinking goals?

Individual Student Practice (Self-Study)

Students can use DebateMind independently to prepare for school debates, competitions, or simply to practise the 21st-century communication skills emphasised by NEP 2020. A recommended self-study routine:

  • Day 1: Generate 3 topics on Beginner difficulty. Save your favourite.
  • Day 2: Use Argument Builder on your saved topic — both Pro and Con sides.
  • Day 3: Use the Debate Timer on Standard mode. Speak aloud during the Speech phase.
  • Day 4: Play 3 rounds of Pro vs Con on Intermediate difficulty.
  • Day 5: Advance to Advanced topics. Track your progress badges.
Section 10

Troubleshooting

IssueSolution
App not loading offlineOpen it once with internet to cache files. After that, it works offline.
Install button not appearingUse Google Chrome or Microsoft Edge. Safari on iOS has limited PWA support.
Progress not savingEnsure browser storage is not set to "Clear on exit". Use the same browser and device.
Timer not audibleThe timer is visual only (toast messages). Ensure notifications are visible on screen.
Author photo not showingEnsure author.jpg is in the same folder as index.html. A placeholder icon shows if not found.
Fonts not loadingGoogle Fonts require internet. Offline, the app uses system serif/sans-serif fallback fonts.
